# Suno Case 03 - Lo-Fi Breakbeat Hybrid

## Brief

- Objectif: hybride lofi-breakbeat memorisable
- BPM cible: 94
- Duree cible: 2:45-3:05
- Interdit: crackle excessif, bas brouillon, loop statique

## Prompt Suno (base)

```
[style: lo-fi breakbeat hybrid, nocturnal, warm, introspective]
[no lyrics]
[tempo: 94 bpm]
[structure: intro, groove section, central contrast break, focused return, fade]
One distinct 4-note motif, warm low-mid body, soft highs, controlled rhythmic motion.
Avoid cliché vinyl noise overload, muddy bass build-up, static looping, glossy pop sheen.
```

## Resultat observe

- Force: vibe differenciante, bonne chaleur
- Faiblesse: motif parfois noye dans la texture

## Correction

Ajouter: "keep lead motif forward and clear in mid frequencies".

## Validation

- [ ] Motif 4 notes clairement identifiable
- [ ] Break central present
- [ ] Duree dans la cible
