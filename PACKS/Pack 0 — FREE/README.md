# Prompting Quickstart Kit

Ce pack gratuit t'aide a passer de prompts vagues a prompts controles en moins de 10 minutes.

## Ce que tu vas trouver

- PROMPT_CHECKLIST.md: la checklist minimale pour cadrer un prompt avant generation.
- PROMPT_MATRIX_LITE.md: une mini matrice pour assembler un prompt vite.
- QUICKSTART_10_MIN.md: la methode d'utilisation la plus rapide.
- EXAMPLES/: trois exemples avant/apres commentes.

## Comment utiliser ce pack

1. Lis QUICKSTART_10_MIN.md.
2. Choisis un element par axe dans PROMPT_MATRIX_LITE.md.
3. Verifie ton prompt avec PROMPT_CHECKLIST.md.
4. Compare avec les exemples si le resultat est trop vague ou trop charge.

## Pour qui

- Debutants qui promptent encore au hasard.
- Musiciens qui veulent un cadre simple sans jargon inutile.
- Createurs qui veulent un systeme agnostique, pas lie a un seul outil.

## Upgrade naturel

- Pack 1 - Starter: version complete du systeme, glossary, guide, matrix complete et mini library.
- Pack 2 - Creator / Studio: framework de production, iteration log, templates projet et mini case studies.
- Pack 3 - Pro / Premium: taxonomie avancee, debugging playbook, standards de versioning et deep cases.

## Regle du jeu

Le but n'est pas de decrire tout le morceau. Le but est de decrire l'intention dominante, la trajectoire et les limites a ne pas franchir.
