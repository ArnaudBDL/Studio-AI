# Mood Atmos Library — Couples Émotion/Atmosphère

Chaque bloc combine un mood (émotionnel) + une atmosphère (sonore). Adapte selon ton objectif.

---

## Warmth + Comfort

**Bloc: Cozy Intimate**
- Mood: cozy, warm, safe
- Atmosphere: intimate, close, organic
- Exemple: "Cozy and intimate mood, warm organic atmosphere, close and personal, gentle but not sterile"

**Bloc: Nostalgic Vintage**
- Mood: nostalgic, melancholic-warm, bittersweet
- Atmosphere: vintage, lo-fi, analog-textured
- Exemple: "Nostalgic and bittersweet mood, vintage atmosphere with tape saturation, warm lo-fi textures, melancholic but not depressing"
