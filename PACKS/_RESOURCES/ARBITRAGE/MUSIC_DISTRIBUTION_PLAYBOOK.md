# Music Distribution Playbook (Post-Production)

Date de mise a jour: 2026-04-25

Objectif: cadrer les canaux de distribution apres production (DSP, radio, playlists, sync, social) pour transformer les morceaux en audience + revenus.

Important:
- Document d'arbitrage interne.
- Les tarifs et conditions changent souvent, verifier avant execution.

## 1) Les 6 canaux a couvrir apres production

1. DSP/streaming stores: Spotify, Apple Music, Deezer, Amazon Music, YouTube Music, TikTok, etc.
2. Social native: TikTok/Instagram/YouTube Shorts via distribution audio + contenus videos courts.
3. Playlists editoriales et independantes: pitching interne + curateurs externes.
4. Radio: radios FM/web radios + radios specialisees + college/community radios.
5. Sync/licensing: bibliotheques musique pour pub, film, contenu UGC, jeux.
6. Direct-to-fan: Bandcamp, site perso, mailing list, drops exclusifs.

### 3.2 Radio: approche terrain

- Radios FM locales et regionales: ciblage genre + geographie.
- Web radios thematiques: plus accessibles pour artistes emergents.
- College/community radios: portes d'entree efficaces pour premiers spins.

Workflow minimum radio:
1. EPK propre (bio courte, pitch track, cover, credits, liens).
2. Version radio edit si necessaire.
3. Mailing segmentee (genre + territoire + format radio).
4. Relance a J+7 et J+14.

## 5) Checklist release-ready (avant distribution)

1. Metadata finalisee: ISRC, UPC, credits, paroles, explicit flag.
2. Artwork valide (format/store rules).
3. Splits et droits confirmes (co-auteurs, producteurs, voix, samples).
4. Politique IA documentee (outil utilise + licence/conditions d'usage).
5. Date de release + retroplanning promo (T-21, T-14, T-7, D-day).
6. EPK + assets courts (vertical video, hooks 10-20s, captions).

## 6) Recommandation operationnelle (version simple)

- Etape 1 (lancement rapide): 1 distributeur principal + 1 outil de pitching curateurs.
- Etape 2 (traction): ajouter radio outreach + 1 campagne micro-budget.
- Etape 3 (scale): ajouter canal sync + optimisation metadata/catalogue.

Stack simple recommande:
- Distribution: DistroKid ou TuneCore
- Pitching: Groover ou SubmitHub
- Suivi: dashboard hebdo (streams, saves, adds playlists, UGC uses)

## 7) KPIs a suivre

- DSP: streams, listeners, save rate, skip rate, source of streams.
- Growth: followers artist profile, adds playlists, retention 28 jours.
- Business: revenu net par morceau, ROI campagne promo, LTV catalogue.
- Radio/promo: taux de reponse, taux d'acceptation, spins/mentions.
