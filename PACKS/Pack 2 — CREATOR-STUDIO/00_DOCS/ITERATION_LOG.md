# Iteration Log

Utilise ce format pour chaque nouvelle generation.

## Template

```md
### Iteration Name

- Date:
- Prompt version:
- Goal of change:
- What changed since previous version:
- Result observed:
- What improved:
- What failed:
- Next targeted change:
```

## Regle

Une iteration utile modifie un nombre limite de variables. Si tu changes tout, tu perds la lecture causale.
