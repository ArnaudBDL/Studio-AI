# Tools Start Here - Creator

Ce guide propose une sequence de travail stable. Les outils restent interchangeables.

## Etape 1 - Cadrage

- Transformer le brief en prompt v1 propre avec les docs du pack.

## Etape 2 - A/B testing

- Comparer des versions avec une seule variable changee.

## Etape 3 - Iteration log

- Formaliser le diagnostic de chaque iteration dans ITERATION_LOG.

## Etape 4 - Lyrics

- Produire des variations de ton et point de vue avec ton assistant de redaction prefere.

## Regle de stabilite

Ne change pas plus de 1 a 2 variables majeures entre deux iterations.

## Reco pratique

- Outils generation possibles: Suno, Udio.
- Outils formulation possibles: ChatGPT, Claude, Gemini.
- Choisis une combinaison et garde-la constante sur un projet.
