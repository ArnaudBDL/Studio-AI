# Prompt Guide - Starter

## 1. Philosophie

- Un prompt sert a cadrer, pas a micro-gerer.
- La clarte vaut mieux que la longueur.
- Une intention forte vaut mieux que dix adjectifs faibles.

## 2. Ordre recommande

1. Core intent
2. Style et genre
3. Mood et atmosphere
4. Structure et dynamique
5. Instrumentation et texture
6. Vocals et lyrics
7. Constraints

Les premiers elements pesent plus lourd.

## 3. Core intent

Le core intent doit tenir en une phrase.

Exemple:

```txt
An intimate alt-rock song that slowly opens into a stronger emotional chorus.
```

## 4. Style et genre

Utilise des clusters simples.

Bon format:

```txt
alt-rock, atmospheric, melancholic, modern
```

Mauvais format:

```txt
post-shoegaze-indietronica-noise-pop-dream-gaze
```

## 5. Mood et atmosphere

Separer emotion et espace sonore.

Exemple:

```txt
Mood: tense, longing
Atmosphere: nocturnal, urban, expansive
```

## 6. Structure et dynamique

Decris le mouvement du morceau dans le temps.

Exemples utiles:

- slow build with delayed payoff
- verse / chorus contrast with restrained intro
- looping evolution with subtle variation

## 7. Instrumentation et texture

Decris des roles, pas un inventaire complet.

Exemple:

```txt
warm pads, distant guitar swells, restrained drums, soft low-end support
```

## 8. Vocals et lyrics

Toujours preciser:

- presence vocale: none / sparse / dominant
- type de delivery
- ton lyrique si necessaire

Exemple:

```txt
sparse female vocals, introspective lyrics, internal monologue
```

## 9. Constraints

Les exclusions doivent etre courtes et utiles.

Exemple:

```txt
avoid bright pop hooks, no festival drop, no overly polished chorus
```

## 10. Erreurs frequentes et correctifs

- Resultat trop generique
  Ajoute structure ou texture.
- Resultat trop chaotique
  Reduis le nombre de styles.
- Resultat trop proche entre chaque tentative
  Garde le style, change seulement structure ou mood.
- Voix non desirees
  Ecris instrumental only tres explicitement.

## 11. Pattern safe

```txt
[style], [mood], [atmosphere], [structure], [texture], [vocals], [constraints]
```

## 12. Pattern exploration

```txt
experimental [style cluster], unstable mood, evolving structure, unusual transitions, sparse anchors, avoid predictable chorus
```
