# Prompt Library Pro

Bibliotheque de prompts premium organisee pour une reutilisation rapide, propre et versionnee.

## 1) Objectif

- Reutiliser des bases robustes par style sans repartir de zero.
- Accelerer le debug via mappings symptome -> correction.
- Maintenir une coherence multi-projets.

## 2) Organisation recommandee

- `*_v01_base.md`: base de depart par univers
- `*_v02_variant.md`: variante stylistique ou energetique
- `*_RC1.md`: candidat release
- `*_FINAL.md`: version validee

## 3) Contenu minimum d'une fiche

- Direction artistique
- Structure cible
- Prompt principal
- Contraintes et exclusions
- Iteration log
- Debug pre-mapping
- Naming/versioning

## 4) Fiches actuelles

- `LOFI_LOUNGE_v01_base.md`

## 5) Regles pro

- Ne jamais ecraser une version validee.
- Une iteration = une hypothese de correction.
- Les exclusions sont obligatoires sur chaque fiche.
- Chaque fiche doit contenir un critere d'acceptation ecoute.

## 6) Roadmap library

Prochaines fiches prioritaires:

- DARK_HYPNO_125_v01_base
- CINEMATIC_PULSE_112_v01_base
- ORGANIC_HOUSE_122_v01_base

Puis decliner en variants `v02` selon usage (DJ, streaming, sync).
