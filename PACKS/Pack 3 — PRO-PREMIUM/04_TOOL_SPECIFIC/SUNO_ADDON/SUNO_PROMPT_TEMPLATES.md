# Suno — Prompt Templates

Templates testés et validés, organisés par genre.  
Chaque template est prêt à copier-coller et à adapter.

---

## Format général

```
[style cluster], [texture], [BPM], [mood], [vocal note], [structure hint]
```

---

## Techno & Club

### Deep Techno — Minimal
```
deep techno, minimal kick, analog warmth, 130 BPM, hypnotic and tense, no vocals, continuous groove no drop
```

industrial techno, metallic percussion, distorted sub bass, 135 BPM, cold and aggressive, no vocals, hard drop at 1:00

melodic techno, lush pads, arpeggiated synth, 128 BPM, melancholic euphoria, no vocals, build and release structure

## Template Custom Mode (avec balises de structure)

Dans le champ **Style of Music** :
```
dark techno, industrial texture, 132 BPM, no vocals
```

Dans le champ **Lyrics** (balises de structure uniquement) :
```
[intro]
[verse]
[drop]
[verse]
[outro: fade]
```
