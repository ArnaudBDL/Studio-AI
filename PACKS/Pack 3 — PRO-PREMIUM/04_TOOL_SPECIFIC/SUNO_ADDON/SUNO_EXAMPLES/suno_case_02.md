# Suno Case 02 - Cinematic Electro Arc

## Brief

- Objectif: instrumental cinematic-electro pour trailer court
- BPM cible: 112
- Duree cible: 3:00-3:20
- Interdit: choeurs epiques dominants, booms cliches, loudness constant

## Prompt Suno (base)

```
[style: cinematic electro, dark, narrative, textured, modern]
[no lyrics]
[tempo: 112 bpm]
[structure: sparse intro, layered build, single precise impact, breathing retreat, concise final rise]
Textured atmosphere, controlled low-end, supportive pulse, smooth highs.
Avoid trailer cliché booms, dominant epic choirs, festival risers, over-compression.
```

## Resultat observe

- Force: bonne progression dramatique
- Faiblesse: impact central parfois trop massif

## Correction

Ajouter: "impact must be controlled and short, no oversized boom tail".

## Validation

- [ ] Intro lisible < 10s
- [ ] 1 impact central unique
- [ ] Break avec vraie respiration
