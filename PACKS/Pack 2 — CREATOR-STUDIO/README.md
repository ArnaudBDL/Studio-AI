# Studio Framework

Ce pack introduit un process de studio reproductible.

## Contenu

- 00_DOCS: documents de reference et regles internes.
- 01_PROJECT_TEMPLATE: template de projet a dupliquer par track.
- 02_PROMPT_LIBRARY: bibliotheque evolutive par famille.
- 03_EXAMPLES_MINI_CASES: mini cas de travail prompt -> correction.
- 04_CHEATSHEETS: aide rapide assembly et debugging.

## Valeur du pack

Ici, on ne parle plus seulement d'ecrire un bon prompt. On parle d'organiser un workflow: brief, references, arbitrages, versions, export.
