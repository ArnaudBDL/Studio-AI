# Prompt Checklist - Free Version

Utilise cette checklist comme filtre avant generation.

## 1. Intention

- [ ] Puis-je resumer le morceau en une phrase ?
- [ ] Le morceau a-t-il un usage clair: ambient de fond, track cinematic, chanson alt-pop, etc. ?

## 2. Style

- [ ] Ai-je defini 2 a 5 mots de style maximum ?
- [ ] Les styles se renforcent-ils au lieu de se contredire ?

Exemple:

```txt
ambient, cinematic, slow-build, atmospheric
```

## 3. Mood et atmosphere

- [ ] Le mood emotionnel est-il clair ?
- [ ] L'atmosphere sonore est-elle claire ?

Exemple:

```txt
Mood: melancholic, introspective
Atmosphere: vast, ethereal, lonely
```

## 4. Structure

- [ ] Ai-je explique comment le morceau evolue ?
- [ ] Ai-je evite les details de theorie musicale inutiles ?

Exemple:

```txt
slow build with evolving textures
```

## 5. Vocals et lyrics

- [ ] Ai-je precise s'il y a des voix ?
- [ ] Si je veux zero voix, ai-je ecrit instrumental only ?

## 6. Contraintes

- [ ] Ai-je liste 1 a 3 choses a eviter ?
- [ ] Les exclusions sont-elles simples ?

Exemple:

```txt
no bright pop chorus
avoid upbeat rhythm
```

## 7. Sanity check

- [ ] Le prompt tient-il en moins de 120 mots ?
- [ ] L'axe principal apparait-il au debut ?
- [ ] Est-ce que je laisse encore un peu de place a l'IA ?

## TL;DR

```txt
One prompt = one dominant intention
Few strong cues beat many weak cues
Clarity first, detail second
```
