# Tools Start Here - Pro

Ce guide t'aide a standardiser un pipeline multi-projets.

Principe: standardiser la methode avant de standardiser les outils.

## Guides studios IA

- Point d'entree: AI_STUDIO_GUIDES/INDEX.md
- Contenu: 26 fiches avec une lecture claire studio par studio
- Niveau: usage concret + sequence PRO
- Dans chaque fiche: lien officiel, description simple, specialisation, generation gratuite

## Pipeline recommande

1. Cadrage et versioning:
   - formaliser la direction et les versions avec les standards du pack.
2. Generation et comparaison:
   - comparer les generations selon les objectifs de vitesse ou de texture.
3. Debugging:
   - traduire symptomes -> corrections prompt/structure dans DEBUG_PLAYBOOK.
4. Lyrics avancees:
   - appliquer des contraintes narratives et tonales precises.

## Arbitrage rapide

- Priorite vitesse: outil de generation rapide.
- Priorite nuance de texture: outil de generation plus fin.
- Priorite standardisation documentaire: assistant de redaction stable.

## Outils possibles (optionnel)

- Generation musicale: Suno, Udio
- Redaction/diagnostic: ChatGPT, Claude, Gemini

## Avec Suno Add-on

Si tu utilises le module Suno dedie, applique d'abord ses templates, puis reinjecte les resultats dans ton iteration log Pro.

## Regle pro

Un bon pipeline n'utilise pas le plus d'outils possible. Il utilise toujours la meme sequence quand elle produit des resultats fiables.
