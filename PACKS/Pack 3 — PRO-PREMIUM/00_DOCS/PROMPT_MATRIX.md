# Prompt Matrix - Pro

Matrice d'aide a la decision pour choisir rapidement le bon cadrage de prompt selon objectif, risque et niveau de polish.

## 1) Matrix by objective

| Objectif | Focus principal | Contraintes minimales | Exclusions critiques |
|---|---|---|---|
| Demo rapide | Idee + groove | genre, mood, BPM | no over-compression |
| Track streaming | Memorisation + clarte | structure, duree, signature | no harsh highs, no muddy low-end |
| Track DJ set | Intro/outro mixables | BPM, sections en barres | no abrupt transitions |
| Sync/cinematic | Narration + dynamiques | arc dramatique, textures | no cliche trailer hits |

## 2) Matrix by risk

| Symptome | Cause probable | Fix prompt |
|---|---|---|
| Trop generique | Prompt trop large | Ajouter signature + exclusions |
| Trop agressif | Energie non plafonnee | Definir cap energie + breaks |
| Trop pop | Hooks brillants implicites | Exclure chorus pop + bright vocals |
| Trop EDM festival | Build/drop par defaut | Exclure big-room drop + risers agressifs |
| Mix brouillon | Aucune regle de lisibilite | Imposer clarte kick/bass et highs doux |

## 3) Matrix by polish level

| Niveau | Description | Prompt detail level |
|---|---|---|
| L0 - Exploration | Recherche de direction | 1-2 phrases, contraintes minimales |
| L1 - Structuration | Direction validee | Structure + exclusions claires |
| L2 - Production | Sortie exploitable | KPI qualite + format de section |
| L3 - Release | Pret distribution | Controles export + naming/versioning |

## 4) Prompt checkpoints

- Checkpoint A: direction artistique lisible
- Checkpoint B: structure et energie stables
- Checkpoint C: exclusions efficaces
- Checkpoint D: sortie repetable

## 5) Quick recipes

### Recipe club hypnotique
"[Genre] at [BPM], [mood], DJ intro/outro, progressive tension, one signature motif, avoid festival drops and harsh highs."

### Recipe cinematic narratif
"[Genre hybrid], dramatic arc from sparse intro to controlled climax, emotional texture, clear sections, avoid trailer cliches and constant loudness."

### Recipe chill lofi
"Warm lo-fi texture, simple groove, restrained arrangement, soft highs, humanized timing, avoid over-polished sheen."
