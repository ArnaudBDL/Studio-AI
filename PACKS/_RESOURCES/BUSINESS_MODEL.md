# Business Model - Studio AI Packs

**Version** : 1.0  
**Date** : 2026-04-26  
**Statut** : interne / stratégie commerciale  

---

## 1. Philosophie générale

Studio AI Packs suit un modèle de **progression de valeur croissante** avec deux principes clés :

1. **Progression obligatoire** : chaque pack suppose la maîtrise du précédent
2. **Accès transparent** : l'acheteur sait exactement ce qu'il achète et pourquoi

---

## 2. Structure des packs

### Pack 0 - FREE (gratuit)
- **Cible** : découverte, zéro friction
- **Valeur** : comprendre les bases en 10 minutes
- **Investissement utilisateur** : temps seulement
- **Prérequis** : aucun

### Pack 1 - STARTER ($29–49)
- **Cible** : débutants sérieux qui veulent cadre stable
- **Valeur** : système complet, glossaire, matrix, 30 prompts library
- **Prérequis** : avoir lu/compris Pack 0
- **Gain vs Pack 0** : +glossaire, +guide structuré, +matrix complète, +library, +examples
- **Prix justifié par** : 3 documents fondamentaux + library + time-to-competence réduit

### Pack 2 - CREATOR-STUDIO ($79–129)
- **Cible** : producteurs qui veulent workflow reproductible
- **Valeur** : framework projet, iteration log, mini cases, cheatsheets
- **Prérequis** : avoir Pack 1 (ou l'inclure dans l'achat)
- **Gain vs Pack 1** : +template projet, +iteration log pro, +mini cases, +cheatsheets
- **Prix justifié par** : investissement studio, templates, processus documenté

### Pack 3 - PRO-PREMIUM ($199–299)
- **Cible** : studios pro, multi-projets, gouvernance KPI
- **Valeur** : taxonomy avancée, debug playbook, standards versioning, deep cases, multi-canal
- **Prérequis** : avoir Pack 1 + 2 (ou les inclure dans l'achat)
- **Gain vs Pack 2** : +taxonomy avancée, +debug complet, +versioning strict, +deep cases, +distribution pro
- **Prix justifié par** : standardisation studio, reproductibilité, risque réduit, distribution optimisée

## 4. Stratégie tarifaire : Bundle + Discount

### Tarification individuelle (achat par pack)

| Pack | Prix | Cible |
|------|------|-------|
| Pack 0 | $0 | Découverte |
| Pack 1 | $39 | Starter sérieux |
| Pack 2 | $99 | Creator-studio |
| Pack 3 | $249 | Pro-premium |
| **Total à l'unité** | **$387** | — |

### Tarification Bundle (accès progressif)

#### Option A : Bundle Starter (Pack 0 + 1)
- Prix : $49 (vs $39+0 = $39 seul)
- Raison : facilite passage 0→1, inclut accès garantie
- Cible : débutants qui veulent directement le complet

#### Option B : Bundle Creator (Pack 0 + 1 + 2)
- Prix : $129 (vs $39+99 = $138 séparé)
- Réduction : -6% (économie de $9)
- Raison : c'est le paquet "production réelle"
- Cible : musiciens sérieux, petits studios

#### Option C : Bundle Pro (Pack 0 + 1 + 2 + 3)
- Prix : $299 (vs $39+99+249 = $387 séparé)
- Réduction : -23% (économie de $88)
- Raison : c'est l'investissement complet, economies d'échelle
- Cible : studios pro, producteurs multi-projets, équipes

### Stratégie d'upgrade

Si quelqu'un achète Pack 1 seul, puis veut Pack 2 :
- Prix Pack 2 seul : $99
- Prix Pack 2 si déjà Pack 1 : $69 (-$30, upgrade prix)
- Raison : fidélité client, pas de re-payement "fondamentaux"

## 5. Cas d'achat : scénarios client

### Scénario 1 : Découvreur → Starter
```
Jour 1: Télécharge Pack 0 (FREE)
Jour 7: Lis QUICKSTART, teste, valide concept
Semaine 2: Achète Pack 1 ($39)
Raison: Veut système complet, pas bricoler
```

### Scénario 2 : Créateur direct → Bundle Creator
```
Cherche "AI music prompt system"
Lit Pack 0, comprend c'est sérieux
Voit "Bundle Creator" à $129 (vs $138 séparé)
Pense : "pour $9 de moins, j'ai projet template ET iteration log"
Achète Bundle Creator (Pack 0+1+2)
```

### Scénario 3 : Studio pro → Bundle Pro
```
Studio établi, cherche standardiser pipeline
Voit Pack 3 mention "taxonomy avancée + debug playbook + standards versioning"
Comprend : c'est pour us, pas pour débutants
Regarde Bundle Pro : $299 pour tout (vs $387)
Pense : $88 d'économie + progression garantie + fondamentaux OK
Achète Bundle Pro
```

## 7. Règles métier

### Progression logique (NON optional)
- ❌ Impossible d'acheter Pack 3 sans Pack 1 + 2
- ✅ Possible d'acheter Pack 2 seul SI on explique "tu vas devoir relire certaines sections de Pack 1"
- ✅ Possible d'acheter Pack 1 seul (c'est l'entry point sérieux)

### Versioning + Support
- Pack 0 : gratuit, support communautaire
- Pack 1 : support email (24h), updates incluses
- Pack 2 : support email + Slack/Discord, updates + feedback
- Pack 3 : support prioritaire, updates + design sprint (optional)

### Contrats de valeur
```
Pack 1 : "Tu apprendras à écrire un prompt cohérent sans surcharge"
Pack 2 : "Tu standardiseras ton workflow et tu itereras proprement"
Pack 3 : "Tu débuggeras systématiquement et tu scaleras à plusieurs projets"
```

Chaque promise = testée avant déploiement.

### Scénario optimiste (300 acheteurs)

- 30 × Bundle Pro : 30 × $299 = $8,970

## 9. Risk mitigation

| Risque | Impact | Mitigation |
|--------|--------|-----------|
| Client achète Pack 3 seul, perd dans fondamentaux | Élevé | Bloquer vente, imposer Bundle |
| Confusion pricing (trop d'options) | Moyen | Recommander Bundle par défaut, option à l'unité visible mais déconseillée |
| Client regrette après achat Pack 1, veut direct Pack 3 | Faible | Upgrade pricing transparent, pas de duplication de coûts |
| Piratage/partage Pack 2/3 | Moyen | Watermark + license per-person dans PDF |
| Support overwhelming | Moyen | Tier par pack (Pack 1 = email, Pack 3 = priorité) |
