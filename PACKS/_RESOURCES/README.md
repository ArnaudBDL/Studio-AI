# PACKS Resources

Ce dossier regroupe les ressources internes de pilotage des packs.

Non destine a la livraison client directe.

## Structure

- ARBITRAGE/
  - AI_TOOL_INDEX.md: index des outils IA, arbitrages et regles d'ajout.
  - DISTRIBUTION_PLATFORMS.md: plateformes de vente (fees, fit, popularite relative).
  - MUSIC_AI_STUDIO_LIST.md: panorama IA musique (compatibilite, qualite, gratuite).
  - MUSIC_DISTRIBUTION_PLAYBOOK.md: canaux de distribution musicale post-production (DSP, radio, playlists, sync).
- ROADMAP/
  - PLAN.md: plan de construction, gamme et logique d'upgrade.

## Regles

- Garder ces fichiers orientés decision.
- Mettre a jour les dates de verification quand necessaire.
- Eviter de dupliquer ici le contenu deja present dans les packs livrables.
