# Example 01 - Ambient

## Intention

Create a dark ambient track for solitary late-night listening.

## Prompt

```txt
ambient, textural, slow-moving, melancholic, vast and ethereal, slow build with evolving textures, warm synth pads and distant drones, instrumental only, avoid rhythmic grooves and bright tones
```

## Variants

```txt
ambient, introspective, lonely, linear progression without obvious chorus, airy layers with minimal pulse, instrumental only, avoid bright tones
```

```txt
ambient, cinematic, melancholic, stormy and expansive, soft opening into wide climax, deep drones and low-end pressure, instrumental only, no heroic trailer tone
```

## Si l'IA derape

Si le rendu devient trop vide, ajoute une structure plus claire ou une texture foreground.
